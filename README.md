# Финансовый Telegram бот

Бот для ведения учета доходов, расходов и накоплений в Telegram.

## Функционал

- 💰 Добавление доходов по категориям
- 💸 Добавление расходов по категориям  
- 🏦 Управление накоплениями (пополнение/снятие)
- 📊 Аналитика с графиками за разные периоды
- 📋 Просмотр истории транзакций
